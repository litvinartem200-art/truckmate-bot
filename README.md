# GidTrack Bot 🚛

Универсальный помощник дальнобойщика в Европе.

## Функции
- 🗺️ Расчёт маршрута и стоимости
- ⛽ Цены на дизель по Европе
- 🛂 Документы для границы (CH, DE, IT)
- 🅿️ Стоянки TIR
- 📋 Правила вождения ЕС
- 🌍 4 языка: UA / RU / FR / EN

## Запуск на Railway
1. Создай проект на railway.app
2. Добавь переменную BOT_TOKEN = твой токен
3. Deploy!

## Локальный запуск
```
pip install -r requirements.txt
BOT_TOKEN=your_token python bot.py
```
